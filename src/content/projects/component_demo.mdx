<TextBlock content="
<p>This project demonstrates how to use <strong>reusable components</strong> in MDX files instead of relying on templates. Each component is designed to be composable and flexible.</p>

<h2>What We Built</h2>

<p>We created four main components:</p>

<ul>
  <li><strong>ProjectHeader</strong> - Displays title, blurb, and date</li>
  <li><strong>Gallery</strong> - Creates a responsive grid of images and videos</li>
  <li><strong>CenteredImage</strong> - Shows a single centered image with caption</li>
  <li><strong>TextBlock</strong> - Renders formatted text content</li>
</ul>

<h2>Benefits of This Approach</h2>

<p>Using components instead of templates gives you:</p>

<ul>
  <li><strong>Flexibility</strong> - Mix and match components as needed</li>
  <li><strong>Reusability</strong> - Use the same components across different projects</li>
  <li><strong>Control</strong> - Fine-tune the layout for each project</li>
  <li><strong>Maintainability</strong> - Update components once, affects all projects</li>
</ul>
" />

  <li>Responsive sizing</li>
  <li>Optional caption</li>
  <li>Hover effects</li>
  <li>Rounded corners and shadow</li>

<TextBlock content="
<h2>Gallery Component</h2>

<p>The <code>Gallery</code> component creates a responsive grid that automatically adjusts to different screen sizes. It includes:</p>

<ul>
  <li><strong>Lightbox functionality</strong> - Click any image to view it full-screen</li>
  <li><strong>Keyboard navigation</strong> - Use arrow keys to navigate</li>
  <li><strong>Video support</strong> - Can include videos alongside images</li>
  <li><strong>Captions</strong> - Optional captions for each media item</li>
  <li><strong>Responsive grid</strong> - Automatically adjusts columns based on screen size</li>
</ul>

<p>You can mix images and videos in the same gallery, and the component will handle both types seamlessly!</p>

<TextBlock content="
<h2>TextBlock Component</h2>

<p>The <code>TextBlock</code> component handles all your text content with proper styling. It supports:</p>

<ul>
  <li><strong>Markdown formatting</strong> - Bold, italic, links, etc.</li>
  <li><strong>Headers</strong> - H1, H2, H3 with proper styling</li>
  <li><strong>Lists</strong> - Both ordered and unordered</li>
  <li><strong>Code blocks</strong> - Syntax highlighting ready</li>
  <li><strong>Blockquotes</strong> - For highlighting important text</li>
</ul>

<h3>Custom Styling</h3>

<p>Each component can be customized with additional CSS classes or inline styles. The components are designed to work together seamlessly while maintaining their individual flexibility.</p>

<blockquote>
  <p>This approach gives you the best of both worlds: the convenience of templates when you need it, and the flexibility of components when you want full control.</p>
</blockquote>
